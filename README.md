# VoiceFlow — Ovozni Matnga Aylantirish Loyihasi

VoiceFlow — foydalanuvchi nutqini real vaqt rejimida matnga aylantirib beruvchi zamonaviy web-ilova. Loyiha frontend texnologiyalarining eng so'nggi imkoniyatlaridan foydalangan holda, foydalanuvchiga yuqori sifatli va qulay interfeysni taqdim etadi.

## Loyiha haqida qisqacha

Loyiha brauzerning ichki **Web Speech API** texnologiyasi asosida qurilgan. Bu foydalanuvchiga hech qanday qo'shimcha pullik xizmatlarsiz yoki tashqi API kalitlarisiz ovozni matnga aylantirish imkonini beradi. Ilova ayniqsa o'zbek tili nutqini tanib olishda yuqori aniqlikka ega.

## Asosiy Imkoniyatlar

- **Real-vaqtda tanib olish:** Nutqni gapirish jarayonida darhol matn ko'rinishida ekranga chiqarish.
- **Glassmorphism Dizayn:** Shaffof va zamonaviy interfeys, foydalanuvchi uchun yoqimli vizual tajriba.
- **Silliq Animatsiyalar:** Interaktiv mikro-animatsiyalar va o'tish effektlari.
- **Tarix (History):** Avvalgi yozib olingan matnlarni saqlash va ularni ko'zdan kechirish imkoniyati.
- **Nusxalash va O'chirish:** Matnni bir tugma orqali clipboardga nusxalash yoki tozalash funksiyalari.

## Texnologik Stack

- **React + TypeScript:** Ishonchli va tezkor dasturiy mantiq.
- **Vite:** Loyihani tezkor yig'ish (build) va ishga tushirish.
- **Framer Motion:** Murakkab va silliq UI animatsiyalar.
- **Lucide React:** Chiroyli va tushunarli piktogrammalar (ikonkalar).
- **Vanilla CSS:** Maxsus va premium ko'rinish uchun moslashtirilgan glassmorphism uslublari.
